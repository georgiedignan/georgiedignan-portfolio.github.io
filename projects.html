Modelling Crib Break Strategies for Haul Truck Circuits

Michael Bell, Anuj Dhorajiwala, Georgie Dignan, Daniel Mactaggart 2021

This project is a business case challenge that is part of the BHP Graduate program. Haulage circuits employ two main strategies for cribbing, these are common crib and hot seating. Cribbing strategies are selected without data driven insights into the comparative performance measures (on prodduction and cost level) for each strategy. The project was create create a tool to model these varying input paramaters with the aim of optimising hot seating strategy to reduce off-circuit travel time.

Pareto Analysis for Unscheduled Downtime Events of Mobile Mine Site Equipment

Georgie Dignan, 2021

Analysis into unscheduled downtime events, to gain a better understanding into why which components are driving the equipment downtime. This assists in failure analysis and helps to prevent future failures of the same cause.
